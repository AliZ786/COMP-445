GET STATEMENTS
---------------------
httpc.py get "http://httpbin.org/get?course=networking&assignment=1"
httpc.py get -v "http://httpbin.org/get?course=networking&assignment=1"
httpc.py get -v -h Content-Type:application/json "http://httpbin.org/get?course=networking&assignment=1"


POST STATEMENTS
---------------------
httpc.py post "http://httpbin.org/post"
httpc.py post -v  "http://httpbin.org/post"
httpc.py post -h Content-Type:application/json "http://httpbin.org/post"
httpc.py post -v -h Content-Type:application/json  "http://httpbin.org/post"
httpc.py post -v --d '{"Assignment": 1}' "http://httpbin.org/post"
httpc.py post -v -h Content-Type:application/json -f test.txt "http://httpbin.org/post"


REDIRECT STATEMENTS
---------------------
